This folder contains a flat arXiv submission package for the public paper.

Files are kept at a single directory level to avoid submission issues with
subdirectories.

Main source:
- `ImpactLLM_paper.tex`

Supporting files:
- `references_ImpactLLM.bib`
- `llncs.cls`
- `splncs04.bst`

Optional local check artifact:
- `ImpactLLM_paper.pdf`
